招生招聘通告

深圳大学光电工程学院拥有光学工程博士点和博士后流动站。根据科研工作需要， “广东省光纤传感技术粤港联合研究中心”拟招聘专职研究人员(研究员/副研究员)和博士后。

研究中心主任王义平是深圳大学首批特聘教授、国家杰青、国家“万人计划”科技创新领军人才。现有团队成员34人；超净实验室500平米，光纤传感专用仪器设备3000余万元。实验室网址：http://cofs.szu.edu.cn。

研究生招生(博士、硕士)

本课题组十分重视对研究生的培养，尤其重视对学生实验能力的锻炼和提升。希望我们的学生喜欢挑战未知问题，并以发表高水平学术论文为乐趣，鼓励学生以科研作为一生追求的目标，支持学生在毕业后出国攻博、出国进行博士后研究或以技术为核心创业。希望学生不习惯把今天的事情拖到明天，喜欢发现问题并解决问题。欢迎志向相投的本科生、硕士生报考本课题组。

诚聘专职研究人员和博士后
深圳大学光电工程学院拥有光学工程一级学科博士点和博士后流动站、“深圳物联网光子器件与传感系统重点实验室”。王义平教授是深圳市政府专项支持的深圳大学首批特聘教授、国家杰青，依托光电工程学院组建光纤传感技术创新团队和实验室。根据科研工作需要招聘博士后研究人员：

研究领域：光纤传感技术及相关领域

应聘条件

  1. 光学工程相关专业，具备光纤传感技术、光电功能材料、光纤通信器件、微纳光子器件等相关方向研究经验。
  2. 应聘专职研究员，应具有国内副教授、国外助理教授或相当任职经历。
  3. 应聘专职副研究员，应为国内外博士毕业生或博士后。
  4. 应聘博士后，应具有或即将获得博士学位，年龄35周岁以下。

专职研究人员待遇

  1. 年薪(税前)：研究员32-42万元、副研究员24-32万元，包括：基本工资、住房补贴和年终绩效。学校缴纳社会保险和住房公积金，提供周转住房、误餐补贴、学术休假、年度体检等；
  2. 符合深圳市后备级人才条件(获国家青年基金、中国博士后基金、或者留深工作5年等条件之一)享受住房补贴160万元(税后)。入选深圳市孔雀计划海外高层次人才(A、B、C三类)，分别享受住房补贴300万、200万、160万元(税后)。博士后出站来深圳签订三年以上合同者可获科研资助30万元；
  3. 论文奖励(一区4万/篇、二区2万/篇、三区1万/篇、四区0.5万/篇)；Nature, Science原刊30万/篇，子刊或影响因子10以上刊物10万/篇。授权国外发明专利2万元/项，中国发明专利1万元/项；
  4. 工作满2年后可申请学校预聘-长聘教师岗位；

博士后待遇

  1. 博士后税后收入25万元/年(工资+政府津贴)，开题后即可领取当年政府津贴12万元(免税)，中期考核后领取第二年政府津贴12万元(免税)。学校缴纳各类社会保险和住房公积金，提供周转住房和误餐补贴；
  2. 泰晤士世界前200名高校(含国内)毕业或博士期间第一作者发表3篇JCR二区以上论文者(33岁以下且获得博士学位1年内)，额外奖励2000元/月；
  3. 论文奖励(一区4万/篇、二区2万/篇、三区1万/篇、四区0.5万/篇)；Nature, Science原刊30万元/篇，子刊或影响因子10以上刊物10万元/篇。授权国外发明专利2万元/项，中国发明专利1万元/项；
  4. 符合深圳市后备级人才条件(获国家青年基金、中国博士后基金、或者留深工作5年等条件之一)享受住房补贴现金160万元。博士后出站后留深圳工作即可获科研资助30万元。优秀博士后可申请留校；
  5. 符合条件者，可申请博士后创新人才支持计划和广东省“珠江人才计划”海外青年人才引进计划(博士后资助项目)
